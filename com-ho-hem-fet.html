¿CÓMO SE HA HECHO?

Qué es este proyecto

Esta es una galería de jugadores del PSG que hice para aprender desarrollo web. Es un proyecto educativo donde puse en práctica lo que sé de HTML, CSS y JavaScript.

La idea era crear algo parecido a las cards de FIFA pero en una página web, mostrando información detallada de cada jugador.

Qué herramientas usé

Para hacer esta galería utilicé estas tecnologías web:

HTML5

Para crear la estructura de todas las páginas

  • Estructura semántica
  • Etiquetas modernas
  • Enlaces entre páginas
CSS3

Para darle estilo y que se vea bien

  • Variables CSS para colores
  • Grid y Flexbox para layout
  • Transiciones y efectos
JavaScript

Para las funciones interactivas

  • Navegación activa
  • Funcionalidades básicas

Cómo lo hice paso a paso

Así fue mi proceso de trabajo:

  1. Planificación: Primero pensé qué quería hacer y cómo organizar la información
  2. Estructura HTML: Empecé creando la página principal con la galería de jugadores
  3. Diseño CSS: Luego le di estilo usando los colores del PSG
  4. Páginas de detalle: Creé una página individual para cada jugador
  5. Navegación: Añadí el menú para moverse entre páginas
  6. Contenido extra: Hice las páginas de referencias y esta explicación

Cómo está organizado el código

El proyecto tiene esta estructura de archivos:

  • index.html - La página principal con la galería
  • com-ho-hem-fet.html - Esta página que explica el proceso
  • referencias.html - Donde pongo todas las fuentes
  • styles.css - Todos los estilos en un solo archivo
  • detalles/ - Carpeta con páginas individuales de cada jugador
  • img/ - Carpeta para las imágenes de los jugadores

Cómo hice el diseño

Para el diseño me basé en:

  • Colores del PSG: Usé el azul (#004170) y rojo (#DA291C) oficiales del equipo
  • Cards de FIFA: Me inspiré en cómo muestran la información de los jugadores
  • Diseño limpio: Quería que fuera fácil de usar y entender
  • Tipografía clara: Usé fuentes que se leen bien en pantalla

En el CSS usé:

  • Variables CSS: Para mantener los colores consistentes
  • CSS Grid: Para organizar las cards en la galería
  • Flexbox: Para alinear elementos dentro de las cards
  • Media queries: Para que se vea bien en diferentes pantallas

Qué cosas hace la página

Las funcionalidades principales son:

  • Galería principal: Muestra todos los jugadores en cards con su información básica
  • Páginas de detalle: Cada jugador tiene su propia página con estadísticas completas
  • Navegación: Menú fijo en la parte superior para moverse entre secciones
  • Diseño adaptable: Se ve bien en computadoras, tablets y móviles
  • Efectos visuales: Las cards tienen efectos al pasar el mouse

Problemas que tuve y cómo los resolví

Algunos desafíos durante el desarrollo:

  • Organizar mucha información: Solución: Crear una estructura clara de HTML y usar CSS Grid
  • Mantener consistencia: Solución: Usar variables CSS para colores y estilos
  • Navegación entre páginas: Solución: Crear un menú reusable y enlaces relativos
  • Contenido de jugadores: Solución: Buscar información confiable en Wikipedia y sitios deportivos

Qué aprendí haciendo esto

Este proyecto me ayudó a:

  • Mejorar mi comprensión de HTML semántico
  • Practicar CSS Grid y Flexbox en un proyecto real
  • Aprender a organizar un proyecto web con múltiples páginas
  • Entender cómo planificar antes de empezar a programar
  • Gestionar contenido extenso de manera organizada

Qué podría mejorar en el futuro

Si tuviera más tiempo, me gustaría añadir:

  • Más interactividad con JavaScript
  • Un sistema de filtros para buscar jugadores
  • Estadísticas en tiempo real
  • Más efectos visuales y animaciones
  • Una versión en otros idiomas
