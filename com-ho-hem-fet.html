¿CÓMO SE HA HECHO?

Qué es este proyecto

Es solo un proyecto que se me asignó en clase, es solo una galería del PSG que hice para aprender. Es un miniproyecto.

Qué herramientas usé

Para hacer esta galería utilicé estas tecnologías web:

HTML5

Para crear la estructura de todas las páginas

  • Estructura semántica
  • Etiquetas modernas
  • Enlaces entre páginas
CSS3

Para darle estilo y que se vea bien

  • Variables CSS para colores
  • Grid y Flexbox para layout
  • Transiciones y efectos

Qué cosas hace la página

Las funcionalidades principales son:

  • Galería principal: Muestra todos los jugadores en cards con su información básica
  • Páginas de detalle: Cada jugador tiene su propia página con estadísticas completas
  • Navegación: Menú fijo en la parte superior para moverse entre secciones
  • Diseño adaptable: Se ve bien en computadoras, tablets y móviles
  • Efectos visuales: Las cards tienen efectos al pasar el mouse

Problemas que tuve y cómo los resolví

Algunos desafíos durante el desarrollo:

  • Organizar mucha información: Solución: Crear una estructura clara de HTML y usar CSS Grid
  • Mantener consistencia: Solución: Usar variables CSS para colores y estilos
  • Navegación entre páginas: Solución: Crear un menú reusable y enlaces relativos
  • Contenido de jugadores: Solución: Buscar información confiable en Wikipedia y sitios deportivos
